# AndroidAutoClicker

一个基于 Android 无障碍服务的简易手机连点器。功能包括悬浮控制条、悬浮球、取点、开始/停止连点、基础间隔、随机等待、坐标随机范围、按下时长、按下偏移和多指同时点击。

## 构建

用 Android Studio 打开本目录，等待 Gradle Sync 完成后运行 `app` 模块。

命令行环境具备 Android SDK 和 Gradle 时也可以执行：

```powershell
gradle :app:assembleDebug
```

## 使用

1. 安装并打开应用。
2. 点击“打开无障碍设置”，启用“手机连点器”服务。
3. 在主界面设置参数并保存：
   - 时间换算：1 秒 = 1000 毫秒，0.5 秒 = 500 毫秒。
   - 基础间隔：每次点击后的固定等待。
   - 随机额外等待：每次在 0 到该值之间随机追加等待。
   - X/Y 随机半径：以取点位置为中心，每次点击随机偏移。
   - 按下时长：一次点击按住多久。
   - 按下偏移：按下到抬起之间的微小移动，默认 0。
   - 同时点击手指数：一次点击同时按下的手指数量。
4. 回到任意应用，使用悬浮控制条的“取点”选择点击中心。
5. 点击“开始”进行连点，再次点击停止。
6. 点击悬浮控制条上的“球”可收起成悬浮球，点击悬浮球可展开控制条。

最低支持 Android 7.0，目标 SDK 35。
